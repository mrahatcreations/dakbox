# Dakbox: আপনার নিজস্ব ওপেন-সোর্স ইমেইল সার্ভার 🚀

Dakbox হলো একটি আধুনিক, দ্রুতগামী এবং ওপেন-সোর্স ওয়েবমেইল প্রজেক্ট, যা তৈরি করা হয়েছে **Stalwart Mail Server** এবং **Vue.js 3** দিয়ে। এটি মূলত তাদের জন্য, যারা গুগল (Google Workspace) বা মাইক্রোসফটের (Microsoft 365) পেছনে প্রতি মাসে টাকা খরচ না করে নিজেদের সার্ভারে একটি প্রিমিয়াম ইমেইল সিস্টেম তৈরি করতে চান।

![Dakbox UI](https://via.placeholder.com/800x400?text=Dakbox+Webmail+UI)

## 🌟 ফিচারসমূহ
- **অবিশ্বাস্য ফাস্ট:** পুরনো IMAP সিস্টেমের বদলে আধুনিক JMAP টেকনোলজি ব্যবহার করা হয়েছে।
- **মডার্ন ডিজাইন:** Vue.js এবং Vuetify দিয়ে তৈরি, যা দেখতে অ্যাপলের সফটওয়্যারগুলোর মতো প্রিমিয়াম ও রাউন্ডেড। 
- **সম্পূর্ণ সেলফ-হোস্টেড:** Docker Compose ব্যবহার করে খুব সহজেই যেকোনো সার্ভারে রান করানো যায়।
- **বিল্ট-ইন সিকিউরিটি:** Stalwart প্যানেল থেকে খুব সহজেই DKIM, DMARC এবং SPF কনফিগার করা যায়, ফলে আপনার ইমেইল কখনো স্প্যামে যাবে না।
- **লাইটওয়েট:** কোনো বিশাল ব্যাকএন্ড নেই, শুধুমাত্র একটি Vue ফ্রন্টএন্ড যা সরাসরি ইমেইল সার্ভারের সাথে কথা বলে।

## 📦 যা যা লাগবে (Prerequisites)
- আপনার কম্পিউটারে বা VPS-এ **Docker এবং Docker Compose** ইন্সটল করা থাকতে হবে।
- একটি নিজস্ব ডোমেইন নেম (যেমন: `dakbox.com`)।

---

## 🛠️ কীভাবে রান করবেন (লোকাল পিসিতে)

১. **প্রজেক্ট ডাউনলোড করুন:**
   ```bash
   git clone https://github.com/mrahatcreative/dakbox.git
   cd dakbox
   ```

২. **ইমেইল সার্ভার চালু করুন:**
   ```bash
   docker-compose up -d stalwart
   ```
   *এটি ব্যাকগ্রাউন্ডে স্টলওয়ার্ট সার্ভার এবং ইমেইলের সব পোর্ট (25, 465, 993) চালু করবে।*

৩. **ফ্রন্টএন্ড বা ওয়েবসাইট চালু করুন:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   এবার ব্রাউজারে `http://localhost:5174` ওপেন করুন। যেহেতু শুরুতে কোনো ইউজার থাকবে না, তাই লগিন পেজের নিচে **"First time setting up? Create Master Admin"** বাটনে ক্লিক করে অ্যাডমিন প্যানেল থেকে আপনার ডোমেইন এবং ইমেইল তৈরি করে নিন।

## 🚀 প্রোডাকশনে লাইভ করা (Coolify / VPS)

Dakbox-কে যেকোনো সার্ভারে ডেপ্লয় করা খুবই সহজ, বিশেষ করে আপনি যদি **Coolify** ব্যবহার করেন:

১. Coolify ড্যাশবোর্ডে গিয়ে আপনার গিটহাব থেকে এই রিপোজিটরিটি সিলেক্ট করুন। 
২. ডেপ্লয়মেন্ট মেথড হিসেবে **Docker Compose** সিলেক্ট করুন।
৩. Coolify-এর সেটিংসে গিয়ে ফ্রন্টএন্ডের জন্য আপনার আসল ডোমেইন (যেমন: `mail.dakbox.com`) সেট করে দিন। (Coolify নিজে থেকেই ফ্রি SSL বা HTTPS বসিয়ে দেবে)। 
৪. Coolify-এর Environment Variables অপশনে গিয়ে আপনার মাস্টার পাসওয়ার্ড ঠিক করে দিন:
   ```env
   ADMIN_MAIL=admin@yourdomain.com
   ADMIN_PASSWORD=আপনার_কঠিন_পাসওয়ার্ড
   ```
৫. ডেপ্লয় বাটনে ক্লিক করুন!
৬. **DNS রেকর্ড:** সবশেষে আপনি যেখান থেকে ডোমেইন কিনেছেন (যেমন Cloudflare), সেখানে গিয়ে আপনার সার্ভারের আইপি অ্যাড্রেসটি A-Record হিসেবে বসিয়ে দিন এবং MX রেকর্ড সেট করুন। 

*একবার লগিন করে মেইন অ্যাকাউন্ট তৈরি করা হয়ে গেলে সিকিউরিটির জন্য Environment Variables থেকে `ADMIN_MAIL` এবং পাসওয়ার্ড মুছে ফেলাই ভালো।*
